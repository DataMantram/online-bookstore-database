# 📚 Online Bookstore Database (PostgreSQL)

This project is a relational database system designed to simulate an online bookstore. It focuses on structured data storage, relationships, and basic data analysis using SQL.

---

## 📌 Overview

The database consists of three main entities:

- **Books** – Stores information about books such as title, author, genre, price, and stock  
- **Customers** – Contains customer details like name, contact, and location  
- **Orders** – Records transactions and connects Customers and Books  

The Orders table acts as a bridge between Books and Customers using foreign keys.
